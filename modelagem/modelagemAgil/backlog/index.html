    Backlog do Produto

    Introdução

    O Backlog do Produto é uma lista contendo todas as funcionalidades desejadas para um produto. O conteúdo desta lista é definido pelo Product Owner.

    O Backlog do Produto é alterado diversas vezes durante o desenvolvimento do projeto, pois com o decorrer do tempo podem aparecer novas funcionalidades a serem implementadas dentro do escopo do projeto, tais funcionalidades podem aparecer a pedido do cliente, ou por nescessidade dos desenvolvedores. Portanto, o backlog não precisa estar totalmente completo no início do projeto, já que irá ocorrer ocasionalmente uma mudança no escopo de tal.

    Metodologia

    Para o desenvolvimento deste artefato, será utilizado a análise das Histórias de Usuário para criar o Backlog do Produto, separando-as em Épicos, e Features, que especificarão de modo mais amplo o que cada uma representa dentro do projeto, e por fim será colocado sua prioridade dentro do projeto de acordo com a prioridade já definida a partir das histórias de Usuário.

    Estrutura do Product Backlog

    Para composição do Product Backlog foi feita a divisão em 3 partes:

    1. Épico : Uma grande parte do trabalho que será fragmentada em pedaços, as chamadas Histórias de Usuário;
    2. Feature : Usados para descrever a funcionalidade em um nível macro;
    3. História de Usuário : Detalhamento mais intrinseco do item.

    Backlog do Produto

    Épico 01 - Conta

    Feature História de Usuário
    FT01 - Cadastro US01 - Eu como usuário gostaria de efetuar login/cadastro com a conta do Google para que eu possa entrar na minha conta e dar continuidade nas atividades.
    FT02 - Logout US08 - Eu como usuário gostaria de sair da minha conta para que eu possa fazer login em outra conta;
    US09 - Eu como usuário gostaria de excluir minha conta para que eu possa desvincular minhas informações do app.
    FT03 - Editar US17 - Eu como usuário gostaria de alterar dados do usuário para que possa manter minha conta atualizada.

    Épico 02 - Perfil

    Feature História de Usuário
    FT04 - Navegar US03 - Eu como usuário gostaria de acessar fórum de suporte do app para que eu possa tirar dúvidas e receber suportes sobre o app;
    US04 - Eu como usuário gostaria de ativar e/ou desativar recebimento de notificações para que eu possa configurar de acordo com minhas preferências;
    US07 - Eu como usuário gostaria de configurar o idioma para que possa escolher o idioma que conheço ou de preferência.

    Épico 03 - Gamificação

    Feature História de Usuário
    FT05 - Melhorias US05 - Eu como usuário gostaria de enviar feedback ao app para que possa melhorar o app;
    US06 - Eu como usuário gostaria de relatar bug do app para que possa melhorar o app.
    FT06 - Desempenho US11 - Eu como usuário gostaria de ver os erros cometidos e tentar refazer para que eu possa melhorar meu desempenho;
    US12 - Eu como usuário gostaria de escolher lições rápidas para que eu possa melhorar meu desempenho;
    US13 - Eu como usuário gostaria de ver o código sendo executado para que eu possa melhorar meu desempenho;
    US14 - Eu como usuário gostaria de pedir dicas para que eu possa melhorar meu desempenho;
    US15 - Eu como usuário gostaria de poder escolher o nível de dificuldade para que eu possa nivelar meu conhecimento;
    US20 - Eu como usuário gostaria de realizar um teste de nivelamento para que eu possa fazer aulas do curso que sejam consistentes com meu conhecimento.
    FT07 - Interação US16 - Eu como usuário gostaria de poder compartilhar uma atividade com amigos para que eu possa comparar meu desempenho;
    US18 - Eu como usuário gostaria de convidar amigos para que eu possa interagir com eles;
    US19 - Eu como usuário gostaria de ver o ranking diário de outros usuários para que eu possa interagir com eles;
    US21 - Eu como usuário gostaria de criar snippet de código para poder compartilha-lo;
    US22 - Eu como usuário gostaria de criar uma comunidade para poder interagir com os amigos.

    Épico 04 - Conteúdo

    Feature História de Usuário
    FT08 - Cursos US02 - Eu como usuário gostaria de navegar pelos cursos no menu de navegação para que eu possa escolher qual curso cursar no momento;
    US10 - Eu como usuário gostaria de escolher a atividade que desejo fazer para que eu possa ter essa liberdade no app.
